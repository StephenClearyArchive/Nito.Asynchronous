In order to build Nito.Async from source, the following tools are required.

For the binaries:
  Visual Studio 2010: http://www.microsoft.com/express/download/

For the help file:
  Sandcastle, May 2008 release (2.4.10520) or newer: http://sandcastle.codeplex.com/releases
  Sandcastle Styles: Sandcastle May 2008 Patch, Rev 2 (2008-11-02) or newer: http://sandcastlestyles.codeplex.com/releases
  HTML Help Workshop: http://msdn.microsoft.com/en-us/library/ms669985.aspx
  Sandcastle Help File Builder, 1.8.0.2 Beta (2009-06-01) or newer: http://shfb.codeplex.com/releases

For the pdb files:
  Debugging Tools for Windows (x86), 6.3 or newer: http://www.microsoft.com/whdc/devtools/debugging/default.mspx
  Perl 5.6 or newer: http://www.activestate.com/activeperl/
  Subversion command-line client: http://www.sliksvn.com/en/download
